# Notify Template - Abar Vault

Use este archivo como base cuando se publique o actualice material relevante para un lead o cliente.

## Metadata

- **Vault path:** `/leads/o-clientes/[slug]/`
- **Fecha de publicacion:** `YYYY-MM-DD`
- **Publicado por:** `Nombre`
- **Estado:** `draft | sent | no-notify`
- **Audiencia:** `ventas | delivery | cliente | interno`
